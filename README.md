# Game Concept:
這是一個模擬跨性別者在台灣更改身分證性別的互動遊戲。使用者需通過三個關卡，每個關卡代表一個現實困境。

# Content Structure:
1. **遊戲首頁**: 標題、簡短文案「為了換一張身分證，你願意付出多少代價？」，以及「開始挑戰」按鈕。
2. **第一關：醫療高牆**: 
   - 描述：你來到醫院精神科，希望能取得診斷證明。
   - 選項 A：配合醫師要求，接受長達兩年的評估（耗時長，壓力增加）。
   - 選項 B：尋找友善醫師（需要運氣，名額稀少）。
3. **第二關：強制手術的代價**:
   - 描述：法律規定必須摘除性器官才能換證。
   - 選項 A：借貸支付巨額手術費並承擔健康風險（負債、身體受損）。
   - 選項 B：因為身體或經濟因素無法手術（換證失敗，遊戲結束/重啟）。
4. **第三關：社會與家庭的目光**:
   - 描述：在職場/換發證件櫃檯，證件與外貌不符被質疑。
   - 選項：承受心理壓力或選擇隱藏自我。
5. **結果頁面**: 根據選擇給予評價，並帶出伴盟的訴求「換證免術、還我基本權」，提供「捐款支持」與「分享」按鈕。

# Technical Requirements:
1. 使用 HTML5, CSS3 (Tailwind CSS 框架), 以及原生 JavaScript。
2. 採用「單頁式設計」，透過 JS 切換顯示區塊（Section），不需跳轉頁面。
3. 視覺風格：簡約、深色調背景、帶有一點壓抑感但充滿希望的文字排版。
4. 必須具備回應式設計 (Responsive Design)，在手機與電腦上都能流暢操作。
5. 加入簡單的進度條 (ProgressBar) 顯示玩家進行到哪一階段。
